PASSPRO AUTOFILL EXTENSION V1.1.0 BETA

Autofill all passwords within your PassPro manager app

How to install the PassPro autofill extension.

Go to edge://extensions or chrome://extensions
enable developer mode.
Press load unpacked and choose the extension folder make sure its extracted first.
Then pin the ext and load it and import your data.


For passwords to autofill you need to have them linked to the website URL. Upon going to the url and opening the ext you can click the password to autofill or select view all passwords.
